# Word Counter

## Description

A simple Python program that reads a text file and displays useful statistics.

## Features

- Read a text file
- Count total characters
- Count total words
- Count total lines
- Count total sentences

## Technologies Used

- Python 3

## Project Structure

```
Word-Counter/
│
├── word_counter.py
├── sample.txt
├── README.md
└── requirements.txt
```

## How to Run

1. Open the project folder.
2. Open the terminal.
3. Run:

```bash
python word_counter.py
```

## Sample Output

```
===== Report =====

Total Characters : 95
Total Words      : 16
Total Lines      : 7
Total Sentences  : 4
```
